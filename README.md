# 🍅 Mommento — Pomodoro + To-Do Web App

🚀 QA Portfolio Project

## PT - Sobre o projeto

O **Mommento** é um web app de produtividade que combina **Pomodoro Timer** com **To-Do List**, permitindo ao usuário gerenciar tarefas e acompanhar seu tempo de foco de forma integrada.

O usuário pode utilizar:

* apenas o Pomodoro
* apenas a lista de tarefas
* ou ambos juntos

Quando uma tarefa está ativa, o tempo de foco é contabilizado para ela. Caso contrário, o tempo é registrado como **tempo de foco no dia**.

---

## ⚙️ Funcionalidades principais

### 🍅 Pomodoro

* Timer de foco
* Pausa de 5 minutos
* Pausa longa de 15 minutos a cada 4 ciclos
* Indicador visual de ciclos (4 bolinhas)
* Alternância manual entre foco e pausa
* Botões:

  * Começar / Pausar
  * Resetar Timer
  * Trocar (foco ↔ pausa)

---

### ✅ To-Do List

* Adicionar tarefas
* Editar tarefas
* Excluir tarefas
* Reordenar tarefas (posição na tela)
* Marcar tarefa como ativa
* Visualização de:

  * tarefas pendentes
  * tarefas concluídas

---

### 📊 Estatísticas

* Horas de foco no dia atual
* Estatísticas de tarefas
* Estatísticas de Pomodoro

---

### 🎨 Interface

* Mudança de cor do tema
* Modo fullscreen
* Interface dinâmica:

  * Frase muda entre foco e pausa
  * Exibição condicional da tarefa ativa
  * Ajuste visual quando não há tarefa

---

## Regras de negócio importantes

* O tempo de foco é associado à tarefa ativa
* Se não houver tarefa ativa → contabiliza como foco geral do dia
* Após 4 ciclos de Pomodoro → pausa longa
* Indicadores visuais refletem progresso dos ciclos

## EN - About the project

**Mommento** is a productivity web app that combines a **Pomodoro Timer** with a **To-Do List**, allowing users to manage tasks and track their focus time in an integrated way.

The user can use:

* only the Pomodoro
* only the task list
* or both together

When a task is active, the focus time is tracked for that task. Otherwise, the time is recorded as **daily focus time**.

---

## ⚙️ Main Features

### 🍅 Pomodoro

* Focus timer
* 5-minute break
* 15-minute long break after 4 cycles
* Visual cycle indicator (4 dots)
* Manual switching between focus and break modes

**Buttons:**

* Start / Pause
* Reset Timer
* Switch (focus ↔ break)

---

### ✅ To-Do List

* Add tasks
* Edit tasks
* Delete tasks
* Reorder tasks (drag & position)
* Mark task as active

**View:**

* Pending tasks
* Completed tasks

---

### 📊 Statistics

* Daily focus hours
* Task statistics
* Pomodoro statistics

---

### 🎨 Interface

* Theme color customization
* Fullscreen mode

**Dynamic interface:**

* Phrase changes between focus and break
* Conditional display of active task
* Layout adapts when no task is active

---

## 🧠 Business Rules

* Focus time is associated with the active task
* If no task is active → counted as general daily focus time
* After 4 Pomodoro cycles → long break
* Visual indicators reflect cycle progress
